How to add


Run following to import my public key:

sudo pacman-key --keyserver hkp://keys.gnupg.net:80 --recv-keys F042ACB684F4FC76DBBA8C4387B75C673974601F
sudo pacman-key --keyserver hkp://keys.gnupg.net:80 --lsign-key F042ACB684F4FC76DBBA8C4387B75C673974601F


Add this at the end of your /etc/pacman.conf

[jojii]
SigLevel = Required DatabaseOptional
 Server = https://repo.jojii.de/_
